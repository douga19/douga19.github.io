Arborescence d'un document HTML

Le langage HTML est un langage de balises permettant de décrire les éléments (paragraphes, titres, etc) d'une page Web. La mise en forme de la page Web se fait à l'aide du langage CSS.

Voici les principales balises HTML :

  • la balise p pour les paragraphes,
  • les balises h1,...,h6 pour les titres,
  • la balise img pour insérer des images,
  • etc
